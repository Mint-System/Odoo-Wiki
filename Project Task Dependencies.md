## Abhängige Aufgabe hinzufügen

Öffnen Sie eine Aufgabe via *Projekt > Alle Aufgaben > Aufgabe wählen* und zeigen die den Tab *Abhängigkeiten* an. Fügen Sie abhängige Aufgaben hinzu.
