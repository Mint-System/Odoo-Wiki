## Verwendung

### Feeder ID festlegen

Öffnen Sie ein Produkt via _Lager > Produkte > Produkte_ und zeigen Sie den Tab _Allgemeine Informationen_ an. Unter dem Feld _Interne Referenz_ finden das Feld _Feeder_.
