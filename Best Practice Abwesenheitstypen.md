# Best Practice: Abwesenheitstypen

## Ferien als Abwesenheitsypen abbilden

Wir empfehlen, dass Sie für jedes jährliche Feriensaldo einen [Abwesenheitstypen erfassen](Abwesenheitszeiten.md#Abwesenheitstypen%20erfassen).

| Name        | Von        | Bis        |
| ----------- | ---------- | ---------- |
| Ferien 2021 | 01.01.2021 | 31.12.2021 |
| Ferien 2022 | 01.01.2022 | 31.12.2022 |
| Ferien 2023 | 01.01.2023 | 31.12.2023 |
| Ferien 2024 | 01.01.2024 | 31.12.2024 |

Am Ende des Jahres können Sie die [verbleibenden Anwesenheitsansprüche anzeigen](HR%20Holidays%20Remaining%20Leaves.md#Verbleibende%20Anwesenheitsansprüche%20anzeigen) und die [Abwesenheitsansprüche exportieren](Abwesenheitszeiten%20Datenmanagement.md#Abwesenheitsansprüche%20exportieren). Passen Sie die exportierte Datei an und importieren Sie die Ansprüche für das neue Jahr.
