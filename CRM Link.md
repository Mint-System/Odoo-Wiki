## Beschreibung

Auf der Formularansicht von Chancen finden Sie zwei Tabs. Im Tab _Links_ können Sie anderen Chancen verlinken. Jeder Link kann eine Bezeichnung haben. Im Tab _Backlinks_ finden Sie eine Liste von Chancen-Links, die auf diese Chance zeigen.
