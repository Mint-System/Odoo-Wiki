## Verwendung

### Beschreibung hinzufügen

Öffnen Sie ein Produkt via *Lager > Produkte > Produkte* und zeigen Sie den Tab *Allgemeine Informationen* an. Unter dem Feld *Interne Referenz* finden das Feld *Type Description*.
